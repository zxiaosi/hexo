一、Github

1.1 Github简介

  • GitHub是一个面向开源及私有软件项目的托管平台（代码托管服务平台）。
  • 更多详情

二、Gihub使用

2.1 Github官网，注册账号并登录，国内访问较慢

2.2 创建仓库（一个仓库通常用来组织一个单独的项目）

  • 点击 New

  • 完善仓库相关信息，然后点击 Create repository 即可创建成功

2.3 创建新文件

  • 点击 Add file --> 选择 Create new file

  • 输入文件名以及内容

  • 预览效果 Preview --> 创建文章 Commit new file

2.4 创建文件夹

  • 因为Github仓库下不允许有空文件夹，所以在创建文件夹时创建一个空文件，防止创建文件夹失败

  • 输入文件名，然后按 / 进入到下一层文件目录

  • 输入空文件名称（随意）

  • Commit new file

2.5 上传文件

  • 点击 Add file --> 选择 Upload files

  • 选择文件 choose your files --> 提交 Commit chagnes

2.6 修改文件、删除文件

  • 选择需要修改或删除的文件

  • 点击对应的图标，即可修改、删除文件

2.7 克隆文件到本地

  • 方法一和方法二需要Git，方法三下载压缩包

2.8 创建分支

  • 分支是一个可以让你在同一时间工作在同一个仓库的不同版本的方法。
  • 在你的仓库中会有一个默认的名叫“master”的主分支，该分支用来存储你最终确定的版本代码。我们用其他的子分支来进行编辑和更改，确定之后再提交到主分支。
  • 更多详情

2.9 删除仓库

  • Settings

  • 拉到最下面，找到 Delete this repository

  • 根据操作输入 仓库名 以及 密码 即可删除（慎重！慎重！慎重！）
