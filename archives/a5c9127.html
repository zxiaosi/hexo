Gridea个人博客搭建(有手就行)

发表于 2021-04-21 更新于 2021-04-26 分类于 个人博客 阅读次数： 评论数：
本文字数： 815 阅读时长 ≈ 1 分钟

一、Girdea下载

1.1 Gridea官网

1.2 Gridea使用手册

1.3 下载Gridea

  • 点击 立即开始

  • 选择对应版本下载

  • 如果速度慢，可以选择 国内 Gitee 下载源

  • 选择后缀为 exe 文件

1.4 安装Gridea

  • 选择 为所有用户安装

  • 选择安装位置

二、Gridea使用

2.1 修改源文件路径

  • 点击 设置

  • 更改站点源文件路径

2.2 预览网站

  • 点击 预览

  • 效果

  • 注意：每次修改都要重新点击预览，刷新页面不会显示修改的效果

2.3 文章

  • 点击 + 开始写文章

  • 写文章

  • 点击 预览 即可看到效果

2.4 菜单

  • 新建菜单

  • 内部链接

  • 外部链接

  • 效果

  • 菜单顺序，长按可挪动位置

2.5 标签

  • 新建标签

  • 输入标签名，URL自动填充

2.6 主题（每一个配置都需要点击右下角的保存）

  • 基础配置

  • 自定义配置（注意：每个主题的自定义配置不一样）

  • 网页图标

  • 头像配置（和网页图标配置一样）

  • 预览效果

三、部署到GitHub

3.1 注册Github

3.2 Github简单使用

3.3 Github配置

  • 创建仓库

  • 开启 Github Pages

  • 获取令牌

    • 点击 Settings

    • 往下拉找到并点击 Developer settings

    • 点击Personal access tokens，生成token Generate new token

    • 令牌名字Note随意，赋予repo权限，最下面有生成按钮

    • 复制令牌（注意：令牌只显示一次，刷新页面令牌就无了）

3.4 Gridea配置

  • 根据提示填入信息，保存后检测连接

  • 连接成功

  • 同步到远程仓库

  • 同步成功（连接成功后，同步有时会报错，多同步几次就好了）

  • 访问自己的网站：https://yourname.github.io/

四、更换主题

4.1 下载主题

  • 更多主题

  • 下载自己喜欢的主题

  • 下载压缩包

  • 解压文件，找到如图的文件目录，并将这个文件夹放到 ./Gridea/themes/ 下，找不到路径 见2.1

4.2 选择主题

  • 重启Gridea软件

  • 选择主题并保存

  • 配置 自定义配置 （有的主题需要先配置 自定义配置 才能预览）

  • 预览效果

五、托管到Vercel（可选）
